# 출석관리 시스템 (Attendance Management)

C# WinForms + MySQL로 만든 출석관리 시스템입니다.  
학생 정보와 출석 상태를 저장하고, 향후 조회 및 통계를 구현할 예정입니다.

## ✅ 기능 요약

- 학생 목록을 콤보박스로 조회
- 출석/지각/결석/조퇴 상태 선택
- 출석 정보 MySQL DB에 저장
